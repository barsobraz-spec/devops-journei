## Что понял

### Процессы
- **Процесс** = запущенная программа с уникальным номером (PID)
- Когда я печатаю команду в bash → bash создаёт **новый процесс** для этой команды
- Когда процесс завершается → его PID исчезает, память освобождается
- Пример: `ps aux` — это отдельный процесс, не сам bash

### SSH
- SSH = зашифрованное соединение между моим Mac и сервером
- Когда я печатаю `ssh root@134.209.234.13` → создаётся процесс bash **на сервере**
- Всё что я печатаю → идёт туда, результат → возвращается обратно

### root vs barsbek
- **root** = суперпользователь (админ). Может делать ВСЁ включая удалить весь сервер
- **barsbek** = обычный пользователь. Linux блокирует опасные действия
- Это защита: если я случайно напечатаю `rm -rf /etc/ssh` → Linux откажет потому что я не root

### sudo
- `sudo` = выполнить команду с правами root, но **только если пароль верный**
- `usermod -aG sudo barsbek` = добавить barsbek в белый список sudo
- Зачем пароль: если рядом сидит кто-то другой → не сможет украсть админ-права
